## 一、系统概述

### 1.1 系统简介

**食材包订阅平台**是一款面向中国市场的智能食材包订阅电商系统。用户可以通过订阅模式定期收到配好的食材包，包含新鲜食材、详细食谱和调料配料。

### 1.2 系统角色

| 角色 | 功能权限 | 适用对象 |
|:---:|:---|:---|
| **普通用户** | 浏览商品、下单购买、管理订阅、维护饮食画像 | 终端消费者 |
| **商家** | 商品管理、订单处理、库存管理 | 食材包供应商 |
| **管理员** | 用户管理、数据统计、系统监控、供应商管理 | 平台运营方 |

### 1.3 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                      前端展示层                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │
│  │   用户端     │  │   商家端     │  │     管理端       │ │
│  └─────────────┘  └─────────────┘  └─────────────────┘ │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                      API 接口层                          │
│         RESTful API（基于 Node.js + Express）            │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                      数据存储层                          │
│              MySQL 数据库 + 本地文件存储                  │
└─────────────────────────────────────────────────────────┘
```

**结算页面操作步骤**：

1. **选择收货地址**：
   - 从已有地址中选择

5. **提交订单**：

## 六、常见问题

### 6.1 登录相关问题

**Q1: 忘记密码怎么办？**

A: 目前系统暂未开放密码找回功能，请联系管理员重置密码。

**Q2: 登录时提示"账号或密码错误"？**

A: 请检查：
- 邮箱地址是否输入正确
- 密码是否包含多余空格
- 大小写是否正确（密码区分大小写）

### 6.2 订单相关问题

**Q3: 订单提交后如何修改收货地址？**

A: 订单提交后地址无法修改。如地址有误，请尽快取消订单后重新下单。

**Q4: 如何查看物流信息？**

A: 进入"我的订单"，点击对应订单进入详情页，在物流信息区域可查看配送进度。

**Q5: 收到商品有问题如何处理？**

A: 进入订单详情页，点击【申请售后】，选择问题类型并提交申请。商家将在24小时内处理。

### 6.3 订阅相关问题

**Q6: 如何修改订阅周期？**

A: 进入"我的订阅"，点击对应订阅进入详情页，选择【修改周期】进行调整。

**Q7: 订阅可以暂停吗？**

A: 可以。在订阅详情页点击【暂停订阅】，已暂停的订阅不会继续扣费和配送。

### 6.4 支付相关问题

**Q8: 支持哪些支付方式？**

A: 目前系统为演示版本，支付功能为模拟支付，点击【确认支付】即可完成支付。

**Q9: 支付失败怎么办？**

A: 请检查网络连接后重新尝试。如多次失败，请联系客服。
