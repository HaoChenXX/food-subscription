1. 系统概述

1.1 平台简介

食材包订阅平台是一个专业的在线食材配送服务平台，为家庭用户、餐饮商家和食材供应商提供一站式的食材采购、配送和管理解决方案。平台采用订阅制商业模式，用户可以根据个人饮食需求选择合适的食材包，享受定期配送服务。

核心价值：
  • 食材新鲜直达，减少中间环节
  • 个性化饮食方案，满足健康需求
  • 定期配送，省时省力
  • 价格透明，品质保障

1.2 技术架构

前端技术栈：

  • React 19 + TypeScript + Vite
  • Tailwind CSS + shadcn/ui组件库
  • Zustand状态管理
  • React Router路由管理

后端技术栈：

  • Node.js 20 + Express框架
  • MySQL 8.0数据库
  • JWT令牌认证
  • RESTful API设计

部署环境：

  • 服务器IP：39.104.25.212
  • 前端端口：80（Nginx代理）
  • 后端端口：3001
  • 数据库：MySQL 3306

1.3 角色定义

平台支持三种用户角色，每种角色拥有不同的权限和功能：

角色 主要功能 适用人群
用户 浏览食材包、下单购买、管理订阅、设置饮食偏好 普通消费者、家庭用户
商家 管理商品、处理订单、监控库存、查看销售数据 食材供应商、餐饮商家
管理员 管理用户、监控平台运营、分析数据、系统配置 平台运营人员、系统管理员

2. 访问与登录

2.1 平台访问

1. 打开浏览器：推荐使用Chrome、Firefox、Edge等现代浏览器

2. 输入地址：在地址栏输入 http://39.104.25.212

3. 等待加载：系统将自动跳转到登录页面

[图1：平台登录页面截图]
截图应显示：平台Logo、登录表单、角色选择标签页、注册链接

2.2 用户登录

2.2.1 选择角色登录

登录页面提供三种角色登录选项：

  1. 用户登录：普通消费者入口
  2. 商家登录：供应商/商家入口
  3. 管理员登录：平台管理员入口
[图2：角色选择标签页截图]
截图应显示：三个标签页（用户、商家、管理员），当前选中用户的登录表单

2.2.2 填写登录信息

1. 选择对应的角色标签页

2. 输入注册邮箱（如：user@example.com）

3. 输入密码（如：user123）

4. 勾选"记住我"（可选）

5. 点击"登录"按钮

登录成功提示：
  • 用户登录后跳转到用户首页
  • 商家登录后跳转到商家管理后台
  • 管理员登录后跳转到管理控制台

2.2.3 忘记密码

如忘记密码，可点击"忘记密码？"链接，系统将发送重置密码邮件到注册邮箱。

2.3 用户注册

如未注册账户，可点击登录页面的"注册"链接进入注册页面。

2.3.1 填写注册信息

1. 基本信息

  • 姓名：填写真实姓名
  • 邮箱：用于登录和接收通知
  • 手机号：用于配送联系
  • 密码：设置6位以上密码
  • 确认密码：再次输入密码

2. 选择角色

  • 用户：普通消费者
  • 商家：食材供应商（需审核）
  • 管理员：仅限内部创建

3. 同意条款

  • 阅读并勾选用户协议
  • 点击"注册"按钮完成注册
[图3：用户注册页面截图]
截图应显示：注册表单、角色选择、用户协议复选框、注册按钮

2.3.2 注册成功

注册成功后系统将：

  1. 自动登录并跳转到对应角色首页
  2. 发送欢迎邮件到注册邮箱
  3. 商家角色需等待管理员审核通过
